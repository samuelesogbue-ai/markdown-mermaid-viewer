# Markdown & Mermaid Viewer

A powerful, modern web application built with Next.js for viewing and exporting Markdown files with integrated Mermaid diagram support.

## üöÄ Features

- **Live Markdown Preview**: Instant rendering of Markdown content.
- **Mermaid Diagrams**: Full support for rendering Mermaid diagrams within your Markdown files.
- **Split Pane View**: Resizable editor and preview panes for a seamless writing experience.
- **Export Options**:
  - **PDF**: Export your rendered Markdown and diagrams to high-quality PDF.
- **Modern UI**: Clean, responsive interface built with Tailwind CSS and Lucide icons.

## üì¶ Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- [pnpm](https://pnpm.io/) (preferred) or npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/samuelesogbue-ai/markdown-mermaid-viewer.git
   cd markdown-mermaid-viewer
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
